# AI Calendar

AI Calendar 是一个以大语言模型驱动的“记忆与规划助理”项目，目标是通过自然语言交互完成过去轨迹记录与未来日程规划。

当前仓库已建立产品需求基线文档，详见 [docs/prd-ai-calendar.md](docs/prd-ai-calendar.md)。

目前也已初始化一版可继续开发的 Web + FastAPI MVP 骨架。

## 当前状态
